# Student Performance Analysis: Mid-Semester Intervention & Predictive Modeling

## 📌 Project Overview
This project addresses the critical challenge of academic "delayed feedback loops." Using a dataset of 1,000 students, this analysis identifies at-risk individuals mid-semester to allow for proactive resource allocation. 

The project specifically solves:
* **The Lag in Intervention:** Identifying struggling students before final exams.
* **Predictive Accuracy vs. Data Noise:** Cleaning erroneous attendance and grade data.
* **The "Compensatory Factor" Mystery:** Quantifying how parental support offsets academic risks.
